Calculations used:

The Original Harris-Benedict Equation


Men (Metric): BMR = 66.5 + ( 13.75 × weight in kg ) + ( 5.003 × height in cm ) – ( 6.755 × age in years )


Men (Imperial): BMR = 66 + ( 6.2 × weight in pounds ) + ( 12.7 × height in inches ) – ( 6.76 × age in years )


Women (Metric): BMR = 655 + ( 9.563 × weight in kg ) + ( 1.850 × height in cm ) – ( 4.676 × age in years )


Women (Imperial): BMR = 655 + ( 4.35 × weight in pounds ) + ( 4.7 × height in inches ) - ( 4.7 × age in years )



~~~~~~~~~~~~


Calorie Intake per day


Sedentary Activity Level (Little or no exercise): Calorie Intake per day = BMR x 1.2


Light Physical Activity Level (1-3 days exercise per week): Calorie Intake per day = BMR x 1.375


Moderate Physical Activity Level (3-5 days exercise per week): Calorie Intake per day = BMR x 1.55


Very Active Physical Activity Level (6-7 days exercise per week): Calorie Intake per day = BMR x 1.725


Vigorous Physical Activity Level (7 or more exercise per week): Calorie Intake per day = BMR x 1.9
